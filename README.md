# Entropy Folding / Eureka Cascade Hypothesis

This repository is a consolidation workspace for the hypothesis and framework spine around entropy folding and eureka cascade theory.

It is intentionally narrow:

- It clarifies existing material.
- It does not extend the theory.
- It does not introduce new examples.
- It preserves uncertainty where the source material is uncertain.

Companion repository

This repository defines the hypothesis, terms, claims, non-claims, formal notes, falsifiability frame, and open uncertainties.

For the companion signal/provenance map, see:

https://github.com/instance001/entropy-folding-cross-domain-signal-atlas

## Scope

This repo is for:

- core hypothesis summary
- claims and non-claims
- key terms needed for a serious outside reader
- model structure and boundaries
- falsifiability, weakening conditions, and limitations already present in source material
- existing methods and tools that may operationalize parts of the hypothesis

This repo is not for:

- proof claims
- cross-domain signal accumulation
- creative synthesis across unrelated sources
- new terminology that is not already grounded in the reference material

## Editorial Rule

If a source is strong, preserve its strength without strengthening it.
If a source is weak or speculative, label it as weak or speculative.
If a relation is unclear, mark it as unclear rather than resolving it creatively.
