# Formalism And Math Notes

## Purpose

Catalogue the mathematical, pseudo-mathematical, and formal-structural material already present in the source ecosystem.

This document is not a place to:

- invent new math
- complete unfinished derivations
- merge separate formalisms into a cleaner system
- smooth over inconsistencies

It is a provenance-preserving catalogue of existing formal scaffolding.

## Formalism Classes Present In The Source Set

The current source corpus appears to contain at least six formalism classes:

1. vector and state-space formalism
2. operational fold-cycle variables and pseudo-equations
3. foundational scope and threshold formalism
4. foundational scale, action, energy, ontic, and stability sketches
5. downstream vault-capacity-routing formalisms
6. topology and cognition-topology language in tooling and architecture materials

## Formalism Status Summary

The formal material in the source ecosystem is real and substantial enough to catalogue separately.

But it should currently be treated as:

- multi-part scaffolding
- uneven in maturity
- partly crisp and partly exploratory
- hypothesis machinery, not signal-atlas evidence

## Placement Rule For The Atlas

Formal material belongs primarily to repo one.

Repo two may reference it only to explain how a candidate signal maps back to the mechanism.

Formalism should not be used in repo two as proof by itself.
