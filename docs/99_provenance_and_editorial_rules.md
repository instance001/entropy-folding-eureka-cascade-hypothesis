# Provenance And Editorial Rules

## Purpose

Make the consolidation method explicit.

## Rules

- Consolidate and clarify existing material only.
- Do not extend the theory.
- Do not add new examples not already present in source material.
- Do not resolve ambiguity creatively.
- If something is uncertain, mark it as uncertain.
- If a claim is strong in the source, preserve it but do not strengthen it.
- If a claim is weak or speculative, label it as weak or speculative.
- Keep source traceability visible.

## Scope Of This Repo

This repository is hypothesis-facing, not atlas-facing.

Cross-domain correspondences belong in the signal atlas repo unless they are needed only as brief orientation context.
